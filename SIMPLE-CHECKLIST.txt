Follow these 3 steps to send emails from your contact form:


┌──────────────────────────────────────────────────────────────────┐
│  STEP 1: GET PUBLIC KEY (1 minute)                               │
└──────────────────────────────────────────────────────────────────┘

  [ ] 1. Go to: https://dashboard.emailjs.com/admin/account
  
  [ ] 2. Login to your EmailJS account
  
  [ ] 3. Find "Public Key" section
  
  [ ] 4. Copy the key (long string)
  
  [ ] 5. Open .env file in your project
  
  [ ] 6. Replace YOUR_PUBLIC_KEY_HERE with your key
  
  [ ] 7. Save the file


┌──────────────────────────────────────────────────────────────────┐
│  STEP 2: CONFIGURE TEMPLATE (3 minutes)                          │
└──────────────────────────────────────────────────────────────────┘

  [ ] 1. Go to: https://dashboard.emailjs.com/admin/templates
  
  [ ] 2. Find template: template_4z4zc8a
  
  [ ] 3. Click Edit
  
  [ ] 4. Settings tab - Fill in:
          From Name: {{from_name}}
          From Email: noreply@emailjs.com
          Reply-To: {{from_email}}
          To Email: setumadhavani@gmail.com
          Subject: Portfolio Contact: {{subject}}
  
  [ ] 5. Content tab - Copy HTML from HOW-TO-SEND-EMAILS.md
  
  [ ] 6. Paste into Content box
  
  [ ] 7. Click Save button
  
  [ ] 8. Wait for "Template saved successfully"


┌──────────────────────────────────────────────────────────────────┐
│  STEP 3: TEST (1 minute)                                         │
└──────────────────────────────────────────────────────────────────┘

  [ ] 1. Open Terminal in VS Code
  
  [ ] 2. Stop server (Ctrl+C)
  
  [ ] 3. Start server: npm run dev
  
  [ ] 4. Open: http://localhost:5173
  
  [ ] 5. Go to Contact page
  
  [ ] 6. Fill out form with test data
  
  [ ] 7. Click Send Message
  
  [ ] 8. See success message
  
  [ ] 9. Check email: setumadhavani@gmail.com
  
  [ ] 10. ✅ Email received!

│  🐛 TROUBLESHOOTING                                              │
└──────────────────────────────────────────────────────────────────┘

  Problem: "Failed to send"
  ├─ Check .env has correct public key
  ├─ Restart server: npm run dev
  └─ Check browser console (F12) for errors

  Problem: Email not arriving
  ├─ Check spam folder
  ├─ Verify email in template: setumadhavani@gmail.com
  └─ Test in EmailJS dashboard first

  Problem: Wrong sender info
  ├─ Check template uses {{from_name}}
  ├─ Check template uses {{from_email}}
  └─ Save template after changes
