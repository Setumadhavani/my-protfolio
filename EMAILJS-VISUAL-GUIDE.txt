┌──────────────────────────────────────────────────────────────────────────────┐
│                         🐛 TROUBLESHOOTING                                   │
└──────────────────────────────────────────────────────────────────────────────┘

    Problem: "Failed to send. Please try again."
    ├─ Check .env file exists
    ├─ Check all three keys are correct
    ├─ Restart dev server
    └─ Check browser console for errors

    Problem: Email not arriving
    ├─ Check spam folder
    ├─ Verify email address in template
    ├─ Check EmailJS service is connected
    └─ Test in EmailJS dashboard first

    Problem: Wrong sender name
    ├─ Check template uses {{from_name}}
    ├─ Not {{user_name}} or hardcoded value
    └─ Save template after editing

    Problem: Can't reply to sender
    ├─ Check Reply-To is set to {{from_email}}
    ├─ Not {{user_email}} or hardcoded value
    └─ Save template after editing
